Websphere

Sobre a oferta

A família de produtos Websphere é formada por servidores de aplicação Java que suportam execução em ambientes on-premisses ou ambientes cloud. Atualmente existem as versões :

  • Liberty (ideal para cloud)
  • Liberty Core - Servidor de aplicações homologado para Web Profile
  • Liberty Base - Servidor de aplicações homologado como full profile
  • Liberty ND - Todos os recursos para construção de aplicações com alta escalabilidade disponíveis

  • Websphere tradicional (tWAS) - Obtém o máximo de ambientes on premisses
  • Websphere Base - Servidor de aplicações full profile sem rec
  • Websphere ND - Servidor de aplicações full profile com recursos sofisticados de controle de escalabilidade

Toda família Websphere mantém o mesmo SLA, com suporte total pela IBM, desde a camada JSE (o JDK da IBM é parte integrante do suporte) até a camada JEE (o próprio servidor de aplicações).
